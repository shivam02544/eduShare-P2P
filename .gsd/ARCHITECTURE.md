## Data Flow

1. User interacts with UI components in `app/` or `components/`.
2. Frontend triggers requests mapping to `app/api/...` endpoint using `fetch()` or `authFetch()` wrapper.
3. API Routes utilize `verifyAuth()` to validate user's Firebase token and checks MongoDB auth records.
4. Business logic interacts with `models/` via `lib/mongodb`. External file storage pushes directly to AWS S3.
5. Responses are returned to the client and React re-renders components using local state.
